Создание заказа

При успешной регистрации заказа будут созданы ссылки на чаты для исполнителя и для заказчика.

Урл запроса

POST /api/v1/marketplace/itelier/order

Тело запроса в формате JSON

Параметры

Описание поля

Тип

atelier_id

ID ательера

строка

atelier_name

Имя/название ательера

строка

atelier_branch_id

ID филиала ательера

строка

atelier_branch_address

Aдрес филиала ательера

строка

client_id

ID покупателя

строка

client_name

Имя/название покупателя

строка

transaction_id

ID заказа

строка

transaction_details

Массив дополнительных параметров заказа

массив

Пример успешного ответа (http code 201)

{
	"status": true,
	"result": {
		"customer_link": "https://wa.me/79267340257?text=order-fc230557-76b6-4b4b-a6ee-cef2c2129e12",
		"merchant_link": "https://t.me/itelier_bot?start=fc230557-76b6-4b4b-a6ee-cef2c2129e12"
	}
}

customer_link

ссылка на WhatsApp чат бот для заказчика

merchant_link

ссылка на Telegram чат бот для исполнителя. Поле присутствует только при создании заказа с новым мерчантом

Пример ответа с ошибкой (http code 400)

{
	"status": false,
	"errors": [
		"ATELIER_NAME_INVALID"
	]
}

Пример ответа с системной ошибкой сервиса без тела (http code 500)

Не регламентирован

Поля ответа

Поля в ответе

Описание поля

status

статус ответа, true / false

result

данные ответа, поле присутствует только в успешном ответе

errors

массив ошибок. Присутствует только при status = false.
