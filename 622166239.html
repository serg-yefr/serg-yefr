BaaS для партнеров : Списание со счёта партнера на незарегистрированную карту с формой / payment_to_unregistered_card_with_form

Created by Наталья Боровикова, last modified by Игорь Бекренёв on февр. 21, 2022

Метод позволяет перевести денежные средства со счета партнера на карту стороннего банка. Возвращает ссылку для ввода реквизитов карты клиента.

У пользователя есть только одна попытка ввести данные при переходе по ссылке.

Время жизни ссылки - 15 минут. 

На карту клиента поступит та сумме, которая будет указана в поле amount. Комиссия за транзакцию будет списана со счета партнера.

Если необходимо переложить комиссию на сторону клиента, укажите это в поле percents_on_user

Для запроса статуса операции используется метод Получение статуса платежа / payment_status

POST /api/v1/refill/{client_id}/unregistered/card/with/form

Пример запроса

{
    "amount": 100,
    "redirect_url": "https://example.com",
    "order_slug": "example_order_id",
    "percents_on_user": "1"
}

{

    "order_slug": "224sdffthg",
    "refill_link": "https://baas-staging.talkbank.io/direct/payment/jg3b743e"
}
