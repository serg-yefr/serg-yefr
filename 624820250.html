BaaS для партнеров : Публичная авторизация карты / сценарий 2.2

Created by Наталья Боровикова, last modified on дек. 29, 2020

  1. Получите токен используя метод - payment_authorization_token 

  2. Передайте полученный токен на клиентскую сторону

  3. Используйте метод Публичный метод авторизации незарегистрированной карты для завершения операции

  4. Дождитесь изменения статуса операции

    1. Получить статус по подписке, мы пришлем уведомление - event_subscription_store/order

    2. Получить статус по запросу - payment_status
