BaaS для партнеров : История операций по счёту партнера / account_transactions

Created by Наталья Боровикова, last modified by Наталья Моргунова on дек. 10, 2021

Метод позволяет запросить транзакции по счету партнера за период.

Расходные операции будут отражены со знаком "-"

GET /api/v1/transactions

URL параметры

Описание

Тип

По умолчанию

fromDate

начиная с даты

string (query)

-1 month

toDate

вплоть до даты

string (query)

now

Пример ответа: Status 200 OK

  {
      "transaction_id": 44443137540,
      "datetime": "2018-01-23 14:03:55",
      "amount": 1500,
      "currency": "RUB",
      "type": "debit",
      "comment": "Card load",
      "status": "approved"
  }

Пример ответа: Status 200 OK

{
      "transaction_id": 44443137541,
      "datetime": "2018-01-24 14:03:55",
      "amount": 1500,
      "currency": "RUB",
      "type": "credit",

      "status": "approved"
  }
