BaaS для партнеров : Отмена операции / payment_reverse

Created by Игорь Бекренёв on июн. 10, 2021

Метод позволяет отменить операцию списания с карты клиента.

После успешного списания перейдет в статус cancelled

POST /api/v1/reverse

Параметры

Описание поля

Тип

order_slug

Уникальный идентификатор операции со стороны партнера

string

Пример ответа

{
  "order_slug": "unique_order_identifier",
  "status": "cancelled"
}

Поле ответа

Описание поля

Уникальный идентификатор операции со стороны партнера

Статус операции
