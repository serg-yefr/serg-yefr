BaaS для партнеров : Используемая терминология

Created by Наталья Боровикова, last modified on дек. 24, 2020

  • barcode —  13 цифр, расположенных под штрих-кодом карты Talkbank

  • Card Ref Id — id - это уникальный номер корреспонденции (направления p2p-перевода) при выполнении p2p перевода.

  • Ref Number — это уникальный номер операции.

  • KYC — операция установления личности клиента, которая выполняется перед проведением финансовой операции.

  • client_id — уникальный идентификатор клиента на стороне партнера. Если партнер не пользуется функционалом для создания клиента, данный параметр будет выдан

  • зарегистрированная карта — карта имеющая barcode (выпуск внутри банка)

  • авторизованная карта — внешняя карта по отношению к банку, но имеющая card ref id

  • незарегистрированная карта — внешняя и неавторизованная карта
