BaaS для партнеров : Установка PIN кода / card_set_pin

Created by Наталья Боровикова, last modified by Наталья Моргунова on янв. 14, 2022

Метод установки PIN кода карты на данный момент работает только для карт, выпущенных банком РФИ.

POST /api/v1/clients/{client_id}/cards/{barcode}/set/pin

URL параметры

Описание поля

Тип

Пример

client_id

Идентификатор клиента

string (path)

client1

barcode

Штриховой код карты

string (path)

31245451

POST параметры

Описание поля

Тип

Пример

pin

4-значный пинкод карты

string

1234

Пример запроса

{
  "pin": "4312"
}

Пример ответа Status 200 OK:

{
 "barcode": "31245451"
}

Пример ответа Status 400 Bad Request:

{
    "description": "This method is available only for RFI cards"
}
