BaaS для партнеров : Получение справки о постановке на учет (снятии с учета) физического лица в качестве самозанятого/ selfemployments_registration_reference

Created by Алексей Кузьмин, last modified by Polina Chunareva on нояб. 08, 2021

GET /api/v1/selfemployments/{client_id}/registration_reference

Параметры строки запроса

Описание поля

Тип

  Примечание

requestYear

Год, за который запрашивается справка

string (path)

В формате YYYY

Пример запроса

GET /api/v1/selfemployments/abc123/registration_reference?requestYear=2021

Пример ответа: Status 200 OK

{
    "client_id": "abc123",
    "status": "success",
    "data": {
        "RegistrationReferencePdf": {
            "mimetype": "application\\pdf",
            "filename": "Справка о постановке на учет в качестве самозанятого.pdf",
            "content": в формате base64"
            }
    }
}

Пример ответа: Status 400 Bad Request

{
    "description": "Bad request",
    "status": "error",
    "errors": [
        "Нет данных за 2020 год. Повторите операцию позже"
    ]
}

{
