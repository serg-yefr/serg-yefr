BaaS для партнеров : Публичный метод авторизации незарегистрированной карты

Created by Наталья Боровикова, last modified by Наталья Моргунова on янв. 25, 2022

POST /client/v1/authorize

POST параметры

Описание поля

Тип

token

Токен, необходимый для запроса к публичному методу авторизации карты. Время жизни - 15 минут. Количество попыток - 1

string

card_info

информация о карте

array


Пример запроса

{
  "token": "4dd1cc4a-ffed-4c69-b3e9-eda3a3673bff",
  "card_info": {
        "number": "4444555566662222",
        "month": "01",
        "year": "2021",
        "cvv": "123",
        "holder": "TALKBANK BLACK"
    }
}

Пример ответа: Status 200 ОК

{
  "charge_link": "http://charge"
}

Пример ответа ошибки токена: Status 403 Forbidden

{
    "description": "Forbidden",
    "errors": [],
    "date": "2021-10-11T08:23:48+00:00"
}

Пример ответа при несуществующем баркоде: Status 404 Not Found

{
    "description": "Requested object is not found"
}

Пример ответа при ошибке метода: Status 500 Internal Server Error

{
    "description": "Runtime error. Please try later."
}
