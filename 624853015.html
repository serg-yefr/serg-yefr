BaaS для партнеров : Cписание денег с авторизованной карты / рекуррентные платежи / сценарий 3.3

Created by Наталья Боровикова, last modified on дек. 29, 2020

  1. Авторизуйте карту методом payment_authorization_with_form или выполните одну из операций списания.

  2. При успехе операции, сохраните card_token 

  3. Настройте логику списаний (по подписке, после выполнения условий и т.д.) и создайте платеж - payment_from_registered_card. 

  4. Дождитесь изменения статуса

    1. Получить статус по подписке, мы пришлем уведомление - event_subscription_store/order

    2. Получить статус по запросу - payment_status
