BaaS для партнеров : Блокировка карты / card_lock

Created by Наталья Боровикова, last modified by Наталья Моргунова on янв. 14, 2022

Метод позволяет заблокировать карту клиента. Блокировка устанавливается в течение 1 минуты.

Для проверки статуса блокировки необходимо запросить метод Получение реквизитов карты / card_details

В поле is_locked будет установлено значение "1".

POST /api/v1/clients/{client_id}/cards/{barcode}/lock

URL параметры

Описание поля

Тип

Пример

client_id

Идентификатор клиента

string (path)

client1

barcode

Штриховой код карты

string (path)
