BaaS для партнеров : Получение платежного поручения / payment_to_account_payment_order

Created by Игорь Бекренёв, last modified on янв. 28, 2022

Платежное поручение формируется в течение 2 часов после получения успешного статуса операцией.

GET /api/v1/account/transfer/{order_slug}/payment-order

Параметры

Описание

Тип

order_slug

Уникальный идентификатор операции со стороны партнера (не более 40 символов)

string (path)

Пример ответа: Status 200 OK

{
  "link": "https:\/\/storage.yandexcloud.net\/baas-payment-orders\/017bd034-32d0-fb18-502b-b46144a6ef79.pdf"
}

Пример ответа ошибки токена: Status 403 Forbidden

{
  "description": "Forbidden",
  "errors": [],
  "date": "2021-10-11T08:23:48+00:00"
}

Пример ответа при недостатке прав: Status 403 Forbidden

{
  "description": "Access Denied."
}

Пример ответа при несуществующем платежном поручении: Status 404 Not Found

{
  "description": "Requested object is not found"
}

Пример ответа при ошибке метода: Status 500 Internal Server Error

{
  "description": "Runtime error. Please try later."
}
