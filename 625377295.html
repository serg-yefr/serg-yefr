BaaS для партнеров : Получение баркода по номеру карты / card_barcode

Created by Наталья Боровикова, last modified by Наталья Моргунова on янв. 14, 2022

Метод возвращает идентификатор карты (barcode). 

Возможные статусы карты: Activated - карта активирована, Pending - карта не активирована.

POST /api/v1/cards/barcode

POST параметры

Описание поля

Тип

Пример

number

номер карты

string (path)

1111222233334444


Пример запроса

{
  "number": "1111222233334444"
}

Пример ответа: Status 200 OK -не подтверждено

200:
{
  "barcode": "2000000000001",
  "status": "Activated"
}

401:
{
  "error": "not found"
}

Пример ответа ошибки токена: Status 403 Forbidden

{
    "description": "Forbidden",
    "errors": [],
    "date": "2021-10-11T08:23:48+00:00"
}

Пример ответа при несуществующем номере карты: Status 404 Not Found

{
    "description": "Requested object is not found"
}

Пример ответа при ошибке метода: Status 500 Internal Server Error

{
    "description": "Runtime error. Please try later."
}

Параметры ответа

Описание поля

Тип

status

Статус карты

string (path)

barcode

Штриховой код карты

string (path)

description

Описание

string (path)
