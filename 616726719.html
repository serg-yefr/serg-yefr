BaaS для партнеров : Публичный метод пополнения незарегистрированной карты / unsigned_payment_to_unregistered_card

Created by Наталья Боровикова, last modified by Наталья Моргунова on февр. 02, 2022

POST /client/v1/refill

POST параметры

Описание поля

Тип

token

Токен, необходимый для запроса к этому публичному методу. Время жизни — 15 минут. Количество попыток — 1

string

card_number

номер карты

string

Пример запроса

{
	"token": "4dd1cc4a-ffed-4c69-b3e9-eda3a3673bff",
    "card_number": "4444111133335555"
}

Пример ответа: Status 200 OK

{
	"token": "4dd1cc4a-ffed-4c69-b3e9-eda3a3673bff"
}

Пример ответа ошибки токена: Status 403 Forbidden

{
    "description": "Forbidden",
    "errors": [],
    "date": "2021-10-11T08:23:48+00:00"
}

Пример ответа при недостатке прав: Status 403 Forbidden

{
    "description": "Access Denied."
}

Пример ответа при ошибочном номере карты: Status 404 Not found

{
    "description": "Requested object is not found"
}

Пример ответа при ошибке метода: Status 500 Internal Server Error

{
    "description": "Runtime error. Please try later."
}
