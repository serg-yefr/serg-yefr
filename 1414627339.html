BaaS для партнеров : Определение ИНН по паспортным данным / selfemployments_inn

Created by Иван Максин, last modified by Наталья Моргунова on дек. 10, 2021

GET /api/v1/selfemployments/inn

URL параметры

Описание поля

Тип

Пример

client_id

Идентификатор клиента

string (path)

client1

Параметры строки запроса

Описание поля

Тип

  Примечание

firstName

Имя

string (path)

lastName

Фамилия

string (path)

patronymicName

Отчество

string(path)

опциональный

birthDay

Дата рождения

string(path)

В формате YYYY-mm-dd

passportSeries

Серия паспорта

string(path)

4 цифры без пробелов

passportNumber

Номер паспорта

string(path)

6 цифр без пробелов

Пример запроса

GET /api/v1/selfemployments/inn?firstName=Иван&lastName=Иванов&birthDay=2000-12-12&passportSeries=1234&passportNumber=123456

Пример ответа: Status 200 OK

{
    "status": "success",
    "data": {
        "Inn": "123456789012"
    }
}

Пример ответа если данные юзера не найдены в ФНС: Status 200 OK

{
    "status": "success",
    "data": {
        "Inn": null
    }
}

Пример ответа при статусе “error“: Status 400 Bad Request

{
    "status": "error",
    "errors": [],
    "description": "Bad request"
}

Пример ответа ошибки токена: Status 403 Forbidden

{
    "description": "Forbidden",
    "errors": [],
    "date": "2021-10-11T08:23:48+00:00"
}

Пример ответа при недостатке прав: Status 403 Forbidden

{
    "description": "Access Denied."
}

Пример ответа: Status 500 Internal Server Error

{
    "description": "Runtime error. Please try later."
}
