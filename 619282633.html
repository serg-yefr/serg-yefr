BaaS для партнеров : Оформление возврата операции / payment_refund

Created by Наталья Боровикова, last modified by Игорь Бекренёв on июн. 10, 2021

Метод позволяет оформить возврат для операций списания с карты клиента.

После успешного возврата операция перейдет в статус cancelled

POST /api/v1/refund

Параметры

Описание поля

Тип

order_slug

Уникальный идентификатор операции со стороны партнера

string

amount

Сумма для возврата.

Если не указана, то будет возвращена вся сумма операции

int

Пример ответа

{
  "order_slug": "unique_refund_identifier",
  "status": "pending"
}

Поле ответа

Описание поля

order_slug

Уникальный идентификатор возврата со стороны партнера

status

Статус операции возврата
