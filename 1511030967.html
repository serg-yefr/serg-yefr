BaaS для партнеров : Работа с самозанятыми / сценарий 6

Created by Алексей Кузьмин, last modified by Polina Chunareva on февр. 02, 2022

1. Создаем клиента

2. Начинаем с проверки статуса смз selfemployments_status

- если статус registered, то переходим к п. 4

- если статус unbound, то направляем запрос на передачу прав Talkbank методом Привязки

- если статус unregistered, для постановки на учет самозанятый должен воспользоваться чат ботом для регистрации здесь

3. Дожидаемся пока смз перейдет в статус registered. Можно опрашивать статус, можно ждать колбэк

4. Регистрируем доход с помощью метода. Если зарегистрировано успешно, то в ответ получите ссылку на чек. Переходим к п.5

5. Запускаем процесс выплаты на карту:

- по номеру карты

- по реквизитам счета

6. Получаем статус выплаты

6.1. запрашиваем статус:

- для выплат по номеру карты

- для выплат по реквизитам счета
