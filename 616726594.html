BaaS для партнеров : Выплата на авторизованную карту / сценарий 4.3

Created by Наталья Боровикова, last modified on дек. 29, 2020

  1. Проведите авторизацию карты с помощью метода payment_authorization или payment_authorization_with_form

  2. Сохраните card_token из метода payment_status

  3. Создайте выплату на авторизованную карту - payment_to_registered_card

  4. Дождитесь изменения статуса

    1. Получить статус по подписке, мы пришлем уведомление - event_subscription_store/order

    2. Получить статус по запросу - payment_status
