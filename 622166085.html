BaaS для партнеров : Получение статуса активации карты / card_activation

Created by Наталья Боровикова, last modified by Наталья Моргунова on янв. 12, 2022

Метод позволяет проверить статус заявки на активацию карты. 

GET /api/v1/clients/{client_id}/cards/{barcode}/activation

URL параметры

Описание поля

Тип

client_id

Идентификатор клиента

string (path)

barcode

Баркод карты

string (path)

Пример ответа: Status 200 OK

{
  "barcode": "2000005405815",
  "status": "activated"
}

Пример ответа ошибки токена: Status 403 Forbidden

{
    "description": "Forbidden",
    "errors": [],
    "date": "2021-11-29T12:49:47+00:00"
}

Пример ответа при ошибке метода: Status 500 Internal Server Error

{
    "description": "Runtime error. Please try later."
}

Пример ответа при ошибке id: Status 404 Not found

{
    "description": "Requested object is not found"
}

Пример ответа при ошибке в баркоде: Status 500 Internal Server Error

{
    "description": "Runtime error. Please try later."
}
