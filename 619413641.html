BaaS для партнеров : Создание заявки на выпуск карты Уралсиб /uralsib_card

Created by Наталья Боровикова, last modified on дек. 25, 2020

Метод запускает выпуск карты указанного типа в банке Уралсиб. Если такой client_id уже существует, то происходит повторный запуск с новыми данными.

POST /api/v1/special/uralsib/cards

Входные параметры

POST параметры

Формат

Описание

client_id

string

Уникальный идентификатор клиента, может состоять из: латинских букв, цифр и символов -._ Не более 40 символов

person.first_name

string

Имя

person.last_name

string

Фамилия 

person.middle_name

string

Отчество

person.phone

string; 

Номер телефона, форматы: +79451111111, 79451111111, 89451111111, 9451111111

person.secret_word

string

Секретное слово

person.gender

int

Пол клиента: 1 - женщина, 0 - мужчина

person.birth_day

date (yyyy-mm-dd)

Дата рождения

person.birth_place

string

Место рождения

person.document.serial

string

Серия паспорта, 4 цифры

person.document.number

string

Номер паспорта, 6 цифр

person.document.date

string

Дата выдачи паспорта

person.document.issuer

string

Кто выдал паспорт

person.document.issuer_code

string

Код подразделения

person.address

string

Адрес регистрации

person.email

string

Электронная почта клиента

bank.branch

string

Код филиала

bank.branch_id

string

Код подразделения

use_courier

int 

Доставить ли карту курьером? 1 - да, 0 - нет (получение в отделении)

delivery.date

date (yyyy-mm-dd)

Желаемая дата доставки (при доставке курьером)

delivery.address

string

Адрес доставки (при доставке курьером)

product_type

string

Тип карты, доступные типы:  profit - карта “Прибыль”
