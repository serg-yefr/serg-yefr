BaaS для партнеров : Списание с незарегистрированной карты на счёт партнера / payment_from_unregistered_card

Created by Наталья Боровикова, last modified by Наталья Моргунова on янв. 27, 2022

Метод используется для списания с карт других банков РФ. 

Метод может быть использован партнерами, которые имеют сертификат PCI DSS. Партнер собирает данные карты клиента на своей стороне и передает их в методе.

Партнер не имеющий возможности хранить данные карты, должен использовать метод Списание с незарегистрированной карты на счет партнера с формой.
Метод возвращает ссылку для подтверждения клиентом операции вводом кода 3ds. 

На счет партнера поступит сумма, указанная в поле "amount" за вычетом комиссии Talkbank.

Для уточнения статуса операции используется метод Получение статуса платежа / payment_status. Запросить статус можно через 1-2 минуты после ввода данных клиентом.

Время жизни ссылки - 15 минут.

Пример запроса

{
    "amount": 100,
    "card_info": {
        "number": "4444111133335555",
        "month": "01",
        "year": "2021",
        "cvv": "123",
        "holder": "TALKBANK BLACK"
    },
    "redirect_url": "https://example.com",
	"order_slug": "example_unique_identifier"
}

Пример ответа: Status 200 OK

{
	"charge_link": "https://baas.talkbank.io/charge/3",
	"order_id": 3,
	"order_slug": "example_unique_identifier"
}

Пример ответа ошибки токена: Status 403 Forbidden

{
    "description": "Forbidden",
    "errors": [],
    "date": "2021-10-11T08:23:48+00:00"
}

Пример ответа при недостатке прав: Status 403 Forbidden

{
    "description": "Access Denied."
}

Пример ответа при ошибочном номере карты: Status 404 Not found

{
    "description": "Requested object is not found"
}

Пример ответа при ошибке метода: Status 500 Internal Server Error

{
    "description": "Runtime error. Please try later."
}
