BaaS для партнеров : Получение списка зарегистрированных чеков / selfemployments_receipts

Created by mikhalev, last modified by Наталья Моргунова on дек. 10, 2021

GET /selfemployments/{client_id}/receipts?dateFrom=date_from&dateTo=date_to

URL параметры

Описание поля

client_id

Идентификатор клиента
пример: client123

date_from

дата начала периода в формате YYYY-MM-DD, без указания времени. время считается 00часов 00минут 00секунд.
Пример: 2021-02-25

date_to

дата окончания периода в формате YYYY-MM-DD, без указания времени. время считается 23часов 59минут 59секунд.

Пример: 2021-02-25
