BaaS для партнеров : Заказ и доставка карты по указанному адресу / card_delivery_store

Created by Наталья Боровикова, last modified on дек. 25, 2020

Метод добавляет новую доставку пользователю. После успешной доставки пользователь должен запустить процесс активации (указать barcode).

Имеется два типа доставки и оплаты:

  • type = courier: доставка осуществляется курьерской службой 

  • type = post: доставка осуществляется Почтой России

POST /api/v1/clients/{client_id}/card-deliveries

URL параметры

Описание поля

Тип

client_id

уникальный идентификатор клиента

string (path)

Описание поля

string

address

Адрес доставки (улица, дом, квартира/офис)

string

Улица

string

Пример входных параметров

{
    "type": "post",
    "phone": "+79260000000",
    "city": "Москва",
    "address": "Кремлевская пл., д. 1",
	"street": "Кремлевская пл.",
	"house": "1",
	"flat": "1",
	"cards_count": 2
}
