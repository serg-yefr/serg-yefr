BaaS для партнеров : Редактирование данных клиента / client_edit

Created by Наталья Боровикова, last modified on дек. 25, 2020

Метод позволяет отредактировать данные созданного ранее клиента и перезапустить процесс идентификации с новыми данными

PUT /api/v1/clients/{client_id}

URL параметры

Описание поля

Тип

client_id

Уникальный идентификатор клиента

string (path)

POST параметры

Описание поля

Тип

person.*

Всё аналогично методу "Создание клиента"

string

Пример входных параметров

{
  person: {
    first_name: "Иван",
    middle_name: "Иванович",
    last_name: "Иванов",
    snils: "116-973-385 89",
	secret_word: "секретный",
    gender: 0,
    phone: "79451111111",
    inn: "6449013711",
    birth_day: "1970-01-01",
    document: {
      serial: "1100",
      number: "111928",
      date: "2015-01-12"
    }
  }
}

Пример ответа

{
  "client_id": "123124"
}

Поле ответа

Описание поля

client_id

Идентификатор клиента
