BaaS для партнеров : Получение токена для списания с незарегистрированной карты / payment_from_unregistered_card_token

Created by Наталья Боровикова, last modified by Наталья Моргунова on янв. 31, 2022

POST /api/v1/charge/{clientId}/token

POST параметры

Описание поля

Тип

redirect_url

Ссылка на страницу, на которую попадет пользователь после ввода 3dsec

string

amount

Сумма (в копейках)

int

Пример запроса

{
   "redirect_url": "https://example.com",
   "amount": 100
}

Пример ответа: Status 200 OK

{
    "token": "066863bf-3e45-4085-b0e4-4979519b9dbb",
    "token_status": "bdf51d04-6ae7-4ef9-ac07-84c9ed615fd2",
    "order_id": 52775,
    "order_slug": "client_wof_35545_nkMmeKX3"
}

Пример ответа ошибки токена: Status 403 Forbidden

{
    "description": "Forbidden",
    "errors": [],
    "date": "2021-10-11T08:23:48+00:00"
}

Пример ответа при недостатке прав: Status 403 Forbidden

{
    "description": "Access Denied."
}

Пример ответа при ошибочном order_id: Status 404 Not found

{
    "description": "Requested object is not found"
}

Пример ответа при ошибке метода: Status 500 Internal Server Error

{
    "description": "Runtime error. Please try later."
}

Поле ответа

Описание поля

Тип

token

Токен, необходимый для запроса к публичному методу списания с незарегистрированной карты. Время жизни - 15 минут. Количество попыток - 1

string

order_id

Уникальный идентификатор операции

int

token_status

Токен, необходимый для запроса к публичному методу получения статуса операции

strting
