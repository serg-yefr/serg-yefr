BaaS для партнеров : Получение статуса доставки / card_delivery_show

Created by Наталья Боровикова, last modified on дек. 25, 2020

Метод позволяет запросить статус заявки на доставку карты

GET /api/v1/clients/{client_id}/card-deliveries/{delivery_id}

URL Параметры

Описание поля

Тип

client_id

Идентификатор клиента

string (path)

delivery_id

Идентификатор доставки

string (path)

Пример ответа

{
  "client_id": "47",
  "delivery_id": 2323,
  "delivery_status_code": 1,
  "delivery_status_description": "new"
}
