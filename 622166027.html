BaaS для партнеров : Описание

Created by Наталья Боровикова, last modified on дек. 24, 2020

Данный API предназначен для автоматизации бизнес-процессов между TalkBank и партнерскими проектами.

В рамках АPI есть возможность:

  • создания новых пользователей;

  • прохождение идентификации клиентом;

  • выпуск пластиковых и виртуальных карт;

  • получения реквизитов карты клиента;

  • получение списка операций по картам;

  • прием платежей (эквайринг);

  • выплаты на карты физ. лиц;

  • холдирование, рекуррентные платежи;

  • p2p переводы.

В качестве основы выбран JSON API. Хост и токены запрашивайте у провайдера этого API.
