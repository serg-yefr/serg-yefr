BaaS для партнеров : Получение списка подписок на события / event_subscription_list

Created by Наталья Боровикова, last modified on дек. 28, 2020

GET /api/v1/event-subscriptions

Пример ответа

{
    "enabled": [
        {
            "id": 1,
            "url": "https://example.com/webhook",
            "event": "payment"
        },
        {
            "id": 2,
            "url": "https://example.com/webhook",
            "event": "activation"
        },
        {
            "id": 3,
            "url": "https://example.com/backoffice/webhook",
            "event": "payment"
        }
    ],
    "available": [
        "identification",
        "activation"
    ]
}

Множество available содержит список типов событий, доступных для подписки.
