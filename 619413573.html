BaaS для партнеров : Выплата на зарегистрированную карту (карту TalkBank) / сценарий 4.4

Created by Наталья Боровикова, last modified on дек. 29, 2020

Для пополнения карт, выпущенных в толкбанк используются следующие методы:

  1. Сохраните barcode карты клиента (после выпуска карты) или запросите список карт клиента - метод card_list

  2. Создайте выплату с использованием идентификатора карты TalkBank (barcode) - метод card_refill

  3. Дождитесь изменения статуса

    1. Получить статус по подписке, мы пришлем уведомление - event_subscription_store/order

    2. Получить статус по запросу - payment_status
