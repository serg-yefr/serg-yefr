BaaS для партнеров : Deptracated: Создание ссылки перевода card2card / card_to_card_payment

Created by Наталья Боровикова, last modified by Наталья Моргунова on дек. 10, 2021

Метод создает ссылку на пополнение или перевод. Требуется указывать либо refId, либо barcode, либо данные карты получателя/отправителя. Параметры from_ean, to_ean, from_card_ref_id, to_card_ref_id являются необязательными.

Для перевода между своими картами необходимо указать параметры to_barcode, from_barcode.

Для перевода на карту стороннего банка необходимо указать параметры from_barcode. Параметр to_barcode = null.

Для пополнения карты  внутри банка необходимо указать параметры from barcode, to_barcode.

Для пополнения карты с карты стороннего банка необходимо указать параметры to_barcode. Параметр from_barcode = null.

При указании параметров отправителя и получателя вместе, по ссылке откроется ссылка на ввод 3dsec.

POST /api/v1/clients/{client_id}/card2card

Пример входных параметров

{
  "from_ean": "20000002232",
  "to_ean": "20003233222",
  "amount": 310,
}


{	
  "amount": 310,
  "card_info": {
     "number": "4444111133335555",
     "month": "01",
     "year": "2021",
     "cvv": "123",
     "holder": "TALKBANK BLACK"
  }
  "to_card_number": "4444111133335551"
}

Пример ответа: Status 200 OK

{
  "client_id": "47",
  "payment_id": "yavz1s",
  "payment_link": "lp.talkbank.io/payment/yavz1s"
}

}
