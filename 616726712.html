BaaS для партнеров : Получение статуса списания с незарегистрированной карты на счёт партнера / payment_from_unregistered_card_status

Created by Наталья Боровикова, last modified by Наталья Моргунова on янв. 27, 2022

Метод выводится из использования. Актуальный метод - Получение статуса платежа / payment_status

Метод позволяет получить статус Списания с незарегистрированной карты на счет партнера. Может быть вызван через 1-2 минуты после ввода данных клиентом

GET /api/v1/charge/unregistered/card/{order_id}

URL параметры

Описание поля

Тип

order_id

Уникальный идентификатор операции

int 

Пример ответа: Status 200 OK

{
  "order_id": 1,
  "is_completed": false,
  "amount": 3000,
  "card_number": "4314********8781",
  "state_code": 10,
  "state_description": "Недостаточно средств"
 }

Пример ответа ошибки токена: Status 403 Forbidden

{
    "description": "Forbidden",
    "errors": [],
    "date": "2021-10-11T08:23:48+00:00"
}

Пример ответа при недостатке прав: Status 403 Forbidden

{
    "description": "Access Denied."
}

Пример ответа при ошибочном order_id: Status 404 Not found

{
    "description": "Requested object is not found"
}

Пример ответа при ошибке метода: Status 500 Internal Server Error

{
    "description": "Runtime error. Please try later."
}
