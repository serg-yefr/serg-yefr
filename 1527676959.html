BaaS для партнеров : Получение информации по доходу СМЗ за период/ selfemployments_income

Created by Алексей Кузьмин, last modified by Polina Chunareva on нояб. 08, 2021

GET /api/v1/selfemployments/{client_id}/income

Параметры строки запроса

Описание поля

Тип

  Примечание

taxPeriod

Год и месяц налогового периода

string (path)

В формате YYYYMM

Пример запроса

GET /api/v1/selfemployments/abc123/income?taxPeriod=202110
