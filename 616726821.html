BaaS для партнеров : Получение списка карт клиента / card_list

Created by Наталья Боровикова, last modified by Наталья Моргунова on дек. 03, 2021

Метод позволяет получить список карты, активированных на клиента

GET /api/v1/clients/{client_id}/cards

URL параметры

Описание поля

Пример

client_id

Идентификатор клиента

client1

Пример ответа: Status 200 OK

[
    "2000007536968",
    "2000007536982",
    "2000007536975"
]

Пример ответа ошибки токена: Status 403 Forbidden

{
    "description": "Forbidden",
    "errors": [],
    "date": "2021-10-11T08:23:48+00:00"
}

Пример ответа при ошибке в id: Status 404 Not found

{
    "description": "Requested object is not found"
}

Пример ответа при ошибке метода: Status 500 Internal Server Error

{
    "description": "Runtime error. Please try later."
}
