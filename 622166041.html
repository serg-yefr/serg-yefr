BaaS для партнеров : Создание клиента и выпуск карты / сценарий 1.1

Created by Наталья Боровикова, last modified on февр. 18, 2022

  1. Создайте клиента с набором данных kyc2 с помощью метода client_store

  2. Когда клиент успешно создан, вызовите метод активации карты: 

    1. Для виртуальной карты - card_activate_virtual

    2. Для пластиковой карты - card_activate

В момент вызова активации карты запускается проверка в банке.

3. Дождитесь изменения статуса активации карты (5-15 мин). Получите статус по запросу, метод - card_activation или оформите подписку метод event_subscription_store/activation

4. Запросите реквизиты карты

  • Для виртуальной карты - вызовите метод card_details (вернется часть данных карты), вызовите метод card_cvv, чтобы отправить оставшиеся данные карты и cvv код клиенту на телефон.

  • Для пластиковой карты - вызовите метод card_details.
