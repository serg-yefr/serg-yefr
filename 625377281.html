BaaS для партнеров : Выпуск виртуальной карты / card_activate_virtual

Created by Наталья Боровикова, last modified by Наталья Моргунова on нояб. 29, 2021

Метод выпускает карту для пользователя и автоматически запускает активацию этой карты. Активация виртуальной карты занимает не более 5 минут.

Максимальное количество карт для пользователя - 5 виртуальных и 5 пластиковых.

POST /api/v1/clients/{client_id}/virtual-cards

URL Параметры

Описание поля

Тип

client_id

уникальный идентификатор клиента

string (path)

Пример ответа: Status 200 ОК

{
    "client_id": "f000225",
    "barcode": "2000007536968",
    "number": "4314090020******",
    "expiration": "10/24"
}

Пример ответа ошибки токена: Status 403 Forbidden

{
    "description": "Forbidden",
    "errors": [],
    "date": "2021-10-11T08:23:48+00:00"
}

Пример ответа при недостатке прав: Status 403 Forbidden

{
    "description": "Access Denied."
}

Пример ответа при несуществующем id клиента: Status 404 Not found

{
    "description": "Requested object is not found"
}

Пример ответа при ошибке метода: Status 500 Internal Server Error

{
    "description": "Runtime error. Please try later."
}
