BaaS для партнеров : Создание ссылки перевода card2card /payment_from_card_to_card

Created by Наталья Боровикова, last modified by Наталья Моргунова on дек. 10, 2021

Метод создает ссылку на пополнение или перевод. Требуется указать полные данные карты отправителя и карту получателя

POST /api/v1/payment/{client_id}/from/card/to/card

URL Параметры

Описание поля

Тип

client_id

Идентификатор клиента

string (path)

POST Параметры

Описание поля

Тип

to_card_number

Номер карты пополнения

string

amount

Сумма в копейках

number

redirect_url

Страница, на которую попадает пользователь после ввода 3dsec

string

card_info

Данные карты отправителя

array

 

card_info.number

Номер карты

string

card_info.month

Месяц окончания срока действия карты

string

card_info.year

Год окончания срока действия карты

string

card_info.cvv

Cvv/cvc код карты

string

card_info.holder

Держатель карты

string

Пример входных параметров

{	
  "amount": 310,
  "card_info": {
     "number": "4444111133335555",
     "month": "01",
     "year": "2021",
     "cvv": "123",
     "holder": "TALKBANK BLACK"
  },
  "to_card_number": "4444111133335551"
}

Пример ответа: Status 200 OK - не подтверждено

{
  "payment_id": "yavz1s",
  "payment_link": "lp.talkbank.io/payment/yavz1s"
}

Пример ответа при ошибке метода: Status 500 Internal Server Error

{
    "description": "Runtime error. Please try later."
}

Пример ответа при ошибке токена: Status 403 Forbidden

{
    "description": "Forbidden",
    "errors": [],
    "date": "2021-12-01T08:00:29+00:00"
}

Поле ответа

Описание поля

Тип

payment_id

Идентификатор ссылки на пополнение

string

payment_link

Конец ссылки на пополнение. Окончание зависит от окружения. baas.talkbank.io → talkbank.io/payment/yavz1s

string
