BaaS для партнеров : Подтверждение/отклонение запроса на привязку Самозанятого к платформе

Created by Polina Chunareva, last modified by Наталья Моргунова on дек. 10, 2021

POST /api/v1/selfemployments/{client_id}/permissions_change/decision

Параметры

Описание

Тип

requestId

Идентификатор запроса Партнера на привязку самозанятого

string (path)

accept

Возможные значение true/false

Пример запроса

{
    "requestId" : "640243",
    "accept": true
}

Пример ответа: Status 200 OK

{
    "client_id": "abc123",
    "status": "success|error",
    "data": {
        "status": "OK|NOK"
    }
}

Пример ответа: Status 200 OK

{
    "client_id": "abc123",
    "status": "error",
    "description": []
}

Пример ответа: Status 400 Not Found

{
    "description": "Requested object is not found"
}

Пример ответа ошибки токена: Status 403 Forbidden

{
    "description": "Forbidden",
    "errors": [],
    "date": "2021-10-11T08:23:48+00:00"
}

Пример ответа при недостатке прав: Status 403 Forbidden

{
    "description": "Access Denied."
}

Пример ответа: Status 500 Internal Server Error

{
    "description": "Runtime error. Please try later."
}
