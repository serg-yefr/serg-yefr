BaaS для партнеров : Получение транзакций по карте / card_transactions

Created by Наталья Боровикова, last modified by Наталья Моргунова on дек. 10, 2021

Метод позволяет партнеру получить историю транзакций по карте клиента.

Расходные операции будут отражаться со знаком "-"

GET /api/v1/clients/{client_id}/cards/{barcode}/transactions

URL Параметры

Описание

По умолчанию

Тип

barcode

Баркод карты

string (path)

fromDate

С какой даты показать историю транзакций

-1 month

ISOdate (query)

toDate

По какую дату показать историю транзакций

now

ISOdate (query)

limit

Лимит записей, максимум 1000

1000

int (query)

page

Страница

1

int (query)


Пример ответа при наличии транзакций: Status 200 ОК

{
      "id": 44443137540,
      "datetime": "2018-01-23 14:03:55",
      "cardNumber": "4738********3296",
      "amount": 1500,
      "cardAmount": 1500,
      "type": "debit",
      "currency": "RUB",
      "category": {
        "name": "Перевод с карты",
        "parent": "Финансы"
      },
      "comment": "TKB CardToCard, MOSCOW",
      "mcc": "6012"
}

Пример ответа при отсутствии транзакций: Status 200 ОК

[]

Пример ответа ошибки токена: Status 403 Forbidden

{
    "description": "Forbidden",
    "errors": [],
    "date": "2021-10-11T08:23:48+00:00"
}

Пример ответа при несуществующем id клиента: Status 200 OK

[]

Пример ответа при несуществующем баркоде: Status 404 Not Found

{
    "description": "Requested object is not found"
}

Пример ответа при ошибке метода: Status 500 Internal Server Error

{
    "description": "Runtime error. Please try later."
}
