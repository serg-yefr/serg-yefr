BaaS для партнеров : Авторизация клиентов

Created by Наталья Боровикова, last modified by Polina Chunareva on июн. 08, 2021

В разделе описаны методы, которые используются для шаблонизирования карты клиента.

Во время операции авторизации с карты клиента списывается и сразу возвращается 1 руб.

В ответ партнер получает уникальный идентификатор карты, который можно использовать для дальнейших операций. 

  • Авторизация карты без формы / payment_authorization

  • Авторизация карты с формой / payment_authorization_with_form

  • Получение токена для авторизации незарегистрированной карты / payment_authorization_token

  • Публичный метод авторизации незарегистрированной карты
