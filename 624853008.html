BaaS для партнеров : Прием платежей по банковской форме / эквайринг / сценарий 3.1

Created by Наталья Боровикова, last modified on дек. 29, 2020

  1. Создайте платеж с помощью метода - payment_from_unregistered_card_with_form

  2. В ответ мы пришлем ссылку на форму ввода реквизитов. Переадресуйте пользователю на эту форму.

  3. Дождитесь изменения статуса операции

    1. Получить статус по подписке, мы пришлем уведомление - event_subscription_store/order

    2. Получить статус по запросу - payment_status

  4. При смене статуса на success операция считается успешной.

  5. При смене статуса на error обратите внимание на код ошибки, его можно показать клиенту, чтобы объяснить причину отказа.
