BaaS для партнеров : Самозанятые в боте / сценарий 5.1

Created by Наталья Боровикова, last modified on дек. 29, 2020

Первая версия работы с сервисом самозанятых. Процесс регистрации проходит внутри бота

  1. Создайте клиента с помощью метода client_store

  2. Передайте клиенту ссылку на бота TB

  3. Подпишитесь на событие selfemployments через event_subscription_store. После того, как пользователь успешного зарегистрируется в качестве самозанятого, вы получите оповещение об этом

  4. Либо можно запросить статус клиента самостоятельно с помощью метода selfemployments_registration_status
