BaaS для партнеров : Проведение упрощенной идентификации клиента / сценарий 1.2

Created by Марина Неделько on февр. 18, 2022

  1. Создайте клиента с набором данных по kyc2 с помощью метода client_store

  2. Дождитесь изменения статуса идентификации клиента (обычно 5 мин)

  • Получить статус по подписке, мы пришлем уведомление - event_subscription_store/identification

  • Получить статус по запросу, метод - client_show

  1. Если идентификация прошла с ошибкой, проверьте правильность переданных данных. Изменить данные по клиенту можно с помощью метода client_edit

  2. При получении статуса identification complete - идентификация считается пройденной успешно
