BaaS для партнеров : Удаление подписки на событие / event_subscription_remove

Created by Наталья Боровикова, last modified on дек. 28, 2020

Позволяет точечно удалить любую подписку по его идентификатору.

DELETE /api/v1/event-subscriptions/{subscription_id}

URL параметры

Описание поля

subscription_id

Идентификатор подписки


После успешного удаления возвращает список event_subscription_list
