BaaS для партнеров : Получение справки о доходах самозанятого/ selfemployments_income_reference

Created by Polina Chunareva, last modified on нояб. 08, 2021

GET /api/v1/selfemployments/{client_id}/income_reference

Параметры строки запроса

Описание поля

Тип

  Примечание

requestYear

Год, за который запрашивается справка

string (path)

В формате YYYY

Пример запроса

GET /api/v1/selfemployments/abc123/income_reference?requestYear=2021

{

            "filename": "Справка о доходе за 2021 год.pdf",

            }
    }
}
