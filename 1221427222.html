BaaS для партнеров : Получения статуса перевода по реквизитам / payment_to_account_status

Created by Polina Chunareva, last modified by Игорь Бекренёв on февр. 04, 2022

GET /api/v1/account/transfer/{order_slug}

Параметры

Описание

Тип

order_slug

Уникальный идентификатор операции со стороны партнера (не более 40 символов)

string (path)

Пример ответа

{
  "completed": true,
  "status": "success",
  "order_slug": "transfer_xxxx",
  "talkbank_commission": 2500,
  "beneficiary_partner_commission": null
}

Поля ответа

Описание

completed

Флаг завершённости операции

status

Статус операции

order_slug

Уникальный идентификатор операции со стороны партнера (не более 40 символов)

talkbank_commission

Комиссия Talkbank

beneficiary_partner_commission

Комиссия партнёра по бенефициару за вычетом комиссии Talkbank
