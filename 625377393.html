BaaS для партнеров : Получение токена для пополнения незарегистрированной карты / payment_to_unregistered_card_token

Created by Наталья Боровикова, last modified by Наталья Моргунова on янв. 31, 2022

POST /api/v1/refill/{clientId}/token

POST параметры

Описание поля

Тип

amount

Сумма (в копейках)

int

order_slug

Ордер (опционально) (не более 40 символов)

string

Пример запроса

{
   "order_slug": "order_slug_1111111111",
   "amount": 100
}

Пример ответа: Status 200 OK

{
    "token": "abee3a2e-c215-4ceb-9515-eec9214ddd75",
    "token_status": "d9b988ae-045c-440d-9cb6-ffee581a515b",
    "order_id": 52776,
    "order_slug": "order_slug_1111111111"
}

Пример ответа ошибки токена: Status 403 Forbidden

{
    "description": "Forbidden",
    "errors": [],
    "date": "2021-10-11T08:23:48+00:00"
}

Пример ответа при недостатке прав: Status 403 Forbidden

{
    "description": "Access Denied."
}

Пример ответа при ошибочном client_id: Status 404 Not found

{
    "description": "Requested object is not found"
}

Пример ответа при ошибке метода: Status 500 Internal Server Error

{
    "description": "Runtime error. Please try later."
}

Поле ответа

Описание поля

Тип

token

Токен, необходимый для запроса к публичному методу пополнения незарегистрированной карты. Время жизни - 15 минут. Количество попыток - 1

string

order_slug

Уникальный идентификатор операции

string

token_status

Токен, необходимый для запроса к публичному методу получения статуса операции

string
