BaaS для партнеров : Получение сводной информацию по лицевому счету ( сумма доходов, вычета)/ selfemployments_account_status

Created by Алексей Кузьмин, last modified by Polina Chunareva on нояб. 08, 2021

GET /api/v1/selfemployments/{client_id}/account_status

Пример запроса

GET /api/v1/selfemployments/abc123/account_status

Пример ответа: Status 200 OK

{
    "client_id": "abc123",
    "status": "success",
    "data": {
        "BonusAmount": "2083.97",
        "UnpaidAmount": "20083.97",
        "DebtAmount": "22083.97"
    }
}

Пример ответа : Status 400 Bad Request

{
    "description": "Bad request",
    "status": "error",
    "errors": [ ]
}

Пример ответа ошибки токена: Status 403 Forbidden

{
    "description": "Forbidden",
    "errors": [],
    "date": "2021-10-11T08:23:48+00:00"
}

Пример ответа при недостатке прав: Status 403 Forbidden

{
    "description": "Access Denied."
}

Пример ответа: Status 404 Not Found

{
    "description": "Requested object is not found"
}

Пример ответа: Status 500 Internal Server Error

{
    "description": "Runtime error. Please try later."
}
