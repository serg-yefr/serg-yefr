BaaS для партнеров : Получение pdf файла для KYC / client_pdf

Created by Наталья Боровикова, last modified on дек. 25, 2020

Используется только с банком Уралсиб. Возвращает специальный pdf для клиента.

GET /api/v1/{client_id}/pdf

URL Параметры

Описание поля

Тип

client_id

Идентификатор клиента

string (path)

barcode

Баркод

string (path)

В ответ вы получите pdf файл в теле ответа ("application/pdf").
