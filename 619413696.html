BaaS для партнеров : Холдирование средств

Created by Наталья Боровикова, last modified on дек. 29, 2020

Методы предназначены для холдирование денежных средств на карте любого банка.

Максимальный срок холдирования денежных средств равен 30 дням.

При наличии у партнера сертификата PCI DSS, можно использовать метод Холдирование без формы. Партнер может собирать платежные данные напрямую у клиента через собственные формы.

При отсутствии сертификата PCI DSS необходимо использовать метод Холдирование с формой.
