BaaS для партнеров : Активация пластиковой карты / card_activate

Created by Наталья Боровикова, last modified by Наталья Моргунова on дек. 07, 2021

Метод запускает активацию карты на пользователя. Может быть как реальной, так и виртуальной.

Параметры POST запроса

Описание поля

Тип

client_id

Идентификатор клиента

string (path)

barcode

Баркод карты (13 цифр)

string (path)

POST /api/v1/clients/{client_id}/cards/{barcode}/activate

Пример ответа: Status 200 ОК

{
    "client_id": "f000225",
    "barcode": "2000006732422",
    "number": "431409******0369",
    "expiration": "07/22"
}

Пример ответа при ошибке метода: Status 500 Internal Server Error

{
    "description": "Runtime error. Please try later."
}

Пример ответа при ошибке метода: Status 403 Forbidden 

{
    "description": "Access Denied."
}

 Пример ответа ошибки токена: Status 403 Forbidden

{
    "description": "Forbidden",
    "errors": [],
    "date": "2021-10-11T08:23:48+00:00"
}

Пример ответа при ошибке id: Status 404 Not found

{
    "description": "Requested object is not found"
}

Пример ответа при ошибке в баркоде: Status 404 Not found

{
    "description": "Requested object is not found"
}
